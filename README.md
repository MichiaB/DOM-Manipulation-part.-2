# DOM Manipulation Part 2

## Description
This lab is the second part of a two-part assignment focused on using JavaScript DOM Manipulation techniques to create dynamic and interactive webpage elements. The project demonstrates event-driven programming by building interactive menu and submenu navigation systems entirely through JavaScript.

---

## Features

- Created a dynamic menu bar using DOM Manipulation
- Created a dynamic sub menu bar using DOM Manipulation
- Added menu interactions using event listeners
- Added sub menu interactions using event-driven programming
- Practiced working with DOM traversal and element creation
- Implemented interactive UI behavior with JavaScript

---

## Technologies Used

- JavaScript
- DOM Manipulation
- HTML
- CSS

---

## What I Learned

- How to dynamically create elements using JavaScript
- How to attach and manage event listeners
- How event bubbling and event delegation work
- How to manipulate styles and classes through the DOM
- How to create interactive navigation systems

---

## Challenges Faced

- Managing event listeners across multiple menu items
- Handling submenu visibility and interactions
- Debugging DOM traversal and targeting issues
